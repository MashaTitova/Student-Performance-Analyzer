# Анализатор успеваемости студентов
## Приложение для анализа успеваемости студентов с расчетом статистики
### В приложении производится:
- Сортировка студентов по среднему баллу, по количеству задолженностей, по фамилии
- Вычисление статистических показателей (среднее арифметическое, медиана, минимальное и максимальное значения)
- Построение рейтингов студентов и групп
- Визуализация данных в виде столбчатых диаграмм и круговых диаграмм
- Поиск студентов по различным критериям с использованием фильтров
- Группировка данных по группам и курсам
- Экспорт отчетов в текстовый файл
### Входные данные:
- Данные об успеваемости 500 студентов по 10 предметам
### Как запустить через Visual Studio
- Склонируйте репозиторий 
- Откройте файл Coursework.var2.sln
- Запустите программу
### Как склонировать репозиторий
- Скопируйте ссылку на репозиторий с GitHub. Ссылка: https://github.com/MashaTitova/Student-Performance-Analyzer.git
- В терминале введите команду git clone <ссылка на репозиторий>
- После выполнения команды в текущей папке появится папка с тем же именем, что и сам репозиторий. Если нужно указать другую папку, в конце команды для клонирования нужно указать её название: git clone <ссылка на репозиторий> <папка>
